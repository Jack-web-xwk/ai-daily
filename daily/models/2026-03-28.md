# AI大模型日报 - 2026年3月28日

## 🔥 今日要闻

**1. Anthropic 下一代超级模型 Claude Mythos 曝光，性能超越 Opus**
据泄露的内部文档，Anthropic 正在秘密测试下一代模型 Claude Mythos（内部代号"Capybara"），其智能水平已全面超越当前旗舰 Claude Opus。公司同时认为该模型带来"前所未有的网络安全风险"，发布节奏或将更为谨慎。
🔗 https://www.aibase.com/news/26616

**2. OpenAI 正式推出 Codex Plugin：一键打包技能与 MCP 配置**
OpenAI 发布 Codex Plugin 服务，支持开发者将 Skills、应用集成和 MCP（模型上下文协议）配置打包为可安装插件，实现团队快速同步。目前已集成 Slack、Figma、Notion、Cloudflare 等工具。上月 Codex 开发者用户已突破 100 万。
🔗 https://www.aibase.com/news/26630

**3. 小米 MiMo-V2 大模型家族正式发布，雷军宣布三年投入 600 亿**
小米发布自研旗舰大模型 MiMo-V2 系列，包括面向 Agent 时代的 MiMo-V2-Pro、全模态大模型 V2-Omni 和语音大模型 V2-TTS。雷军宣布未来三年 AI 领域投入超 600 亿元，2026 年 AI 研发及资本开支已超 160 亿元。首款 AI 原生智能手机进入封测。
🔗 https://www.aibase.com/news/26612

**4. Cohere 开源轻量语音识别模型 Transcribe（20亿参数）**
Cohere 发布开源语音识别模型 Cohere Transcribe，仅 20 亿参数，支持 14 种语言，可直接部署在手机、PC 等终端设备上。在 Hugging Face ASR 排行榜上超越 ElevenLabs Scribe 和 Qwen3，采用 Apache 2.0 协议开源。
🔗 https://www.aibase.com/news/26624

**5. 腾讯发布 Agent 产品全景图，构建 AI 时代全栈引擎**
腾讯首次系统性展示 Agent 产品顶层设计，底层为 AI Infra 智算底座，中层依托 TokenHub 大模型服务平台（集成混元及 MiniMax、DeepSeek、Kimi 等主流模型），上层通过 Skills 工具箱打通微信、企微、QQ 等国民级入口。
🔗 https://www.aibase.com/news/26628

## 💡 观察与趋势

**Agent 生态正在成为各大厂的核心战场** — 从 OpenAI 的 Codex Plugin 到腾讯全景 Agent 架构再到美团的 AI 助手，"大模型→Agent→工具调用"的链条已成共识，插件化、可组合的 Agent 架构正在快速标准化。同时，开源轻量化模型（如 Cohere Transcribe）的涌现表明，端侧部署和隐私友好型 AI 正在成为新的竞争维度。
